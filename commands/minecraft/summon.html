Parent: minecraft

Summon another user.

Syntax: minecraft summon <user>

Permissions

Everyone can run this command.

Arguments

user

The user to summon

Type: User Argument

Examples

An example usage of the command is %minecraft summon @PokestarBot

Subcommands

The command has no subcommands.

Aliases

This command has no aliases.

Slash Commands

  • /summon
