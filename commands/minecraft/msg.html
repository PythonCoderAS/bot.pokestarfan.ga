Parent: minecraft

Message another user.

Syntax: minecraft msg <user> <message>

Permissions

Everyone can run this command.

Arguments

user

The user to message

Type: User Argument

message

The message to send.

Examples

An example usage of the command is %minecraft msg @PokestarBot How's the weather?

Subcommands

The command has no subcommands.
