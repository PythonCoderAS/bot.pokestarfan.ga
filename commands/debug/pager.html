Parent: debug

The pager debug commands are commands that allow the testing of pagers. When called, the parent command shows a pager of subcommands.

Syntax: debug pager

Permissions

Everyone can run this command.

Arguments

The command has no arguments.

Examples

An example usage of the command is %debug pager.

Subcommands

  • lines
  • page

Aliases

This command has no aliases.

Slash Commands

No slash commands link to the command.
