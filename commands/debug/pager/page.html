Parent: debug pager

Generate a pager with a given amount of pages.

Syntax: debug pager page <pages> [--text=True] [--embed=False] [--text_template=This is test line #{number}] [--embed_description_template=This is test embed #{number}]

Permissions

Only the owner can run the command.

Arguments

pages

The number of pages to generate.

--text=

Whether or not to include text. Defaults to False.

Type: Boolean Argument

--embed=

Whether or not to include an embed. Defaults to False.

Type: Boolean Argument

--text_template=

The text that will be generated for each page when the text argument is enabled. By default, the string This is test text #{number} is used. The string can contain variable placeholders that take the form {<variable>}, where <variable> is the name of the variable.

--embed_description_template=

The embed description that will be generated for each page when the embed argument is enabled. By default, the string This is test embed #{number} is used. The string can contain variable placeholders that take the form {<variable>}, where <variable> is the name of the variable.

Examples

An example usage of the command is %debug pager page --text=1 --embed=0.

Subcommands

The command has no subcommands.
