Parent: debug

The root command for testing the bot's exception handling. When called directly, shows a pager of subcommands.

Syntax: debug exception

Permissions

Only the owner can run the command.

Arguments

The command has no arguments.

Examples

An example usage of the command is %debug exception.

Subcommands

  • trigger

Aliases

This command has no aliases.
