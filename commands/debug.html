The debug commands are commands that trigger situations that would be difficult to do otherwise. These situations are used for testing the bot. The parent command just shows the pager of subcommands.

Permissions

Only the owner can run the command.

Arguments

The command has no arguments.

Examples

An example usage of the command is %debug.

Subcommands

  • argument_links
  • command_info
  • eval
  • exception
  • exec
  • pager
  • reload
  • test

Note: This command has additional automatically-created aliases.

Slash Commands
