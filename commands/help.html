Get help on a cog or command. When left blank, brings up a list of all commands you can run in the current channel.

Syntax: help [command]

Permissions

Everyone can run this command.

Arguments

command

The command to get help on. Can also be a cog.

Examples

An example usage of the command is %help dashboard.

Subcommands

The command has no subcommands.

Aliases

This command has no aliases.

Slash Commands

No slash commands link to the command.
